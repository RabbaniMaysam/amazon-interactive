# Interactive calibration — Amazon brand-strategy paper

A single-page browser tool for exploring the equilibrium of the platform-fee /
self-preferencing model used in the paper.

Live page: https://rabbanimaysam.github.io/amazon-interactive/

Drag the sliders for the model parameters ($c$, $F$, $v$, $\alpha$, $\delta$,
$\lambda$, $K$); the equilibrium values, profit decomposition, welfare, and
dynamic transition path update in real time. Defaults match the numerical
calibration reported in Section F of the manuscript.

The page runs entirely client-side. No server, no analytics, no tracking.

## Source

The source of truth lives in the (private) paper repository under
`docs/index.html`. This repository is a deployment mirror.
